How DrawIt Works?

DrawIt uses HTML Canvas to implement the drawing feature. and almost every feature that is available in this app, relys on HTML Canvas. as vHashLabs focus to build Fast - Less Complicated - Convienent Apps, we though to not use too much manual math and hence we used HTML Canvas.
